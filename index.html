Linux 2、1、& 和 >

July 21, 2021

1 分鐘

小記一下 command 2>&1 > /dev/null 和 command > /dev/null 2>&1 的差別。首先先看 2>&1 的意思。在 Linux 的 file descriptor 中，1 表示 stdout (標準輸出) 表示程式執行中輸出訊息的地方，預設為 terminal。而 2 表示 stderr (標準錯誤輸出) 表示程式執行中發生錯誤時輸出錯誤訊息的地方，預設也是 terminal。
